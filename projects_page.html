Computer Vision | Deep Learning

Automatic Number Plate Detection with YOLO and PyTesseract

  • Data Parsing: Extracted crucial information like image paths, dimensions, and bounding box coordinates for number plates using ElementTree for parsing XML annotation files.
  • Image Preprocessing: Employed TensorFlow to read, resize, and convert images into numerical arrays, followed by normalization for model training.
  • Number Plate Localization: Determined the center of each number plate object based on the bounding box information.
  • YOLOv5 Model Training: Leveraged pre-trained YOLOv5 weights to train a custom object detection model for accurately identifying number plates within images.
  • Filtering and Refinement: Applied confidence and probability thresholds to filter detections and utilized non-maximum suppression to eliminate redundant bounding boxes.
  • Region of Interest Extraction: Extracted specific regions containing number plates (ROIs) based on the detected bounding boxes.
  • Optical Character Recognition (OCR): Performed character recognition on the extracted ROIs using the pre-trained PyTesseract OCR model.
  • Text Extraction: Extracted the text information from the recognized characters using the PyTesseract "image_to_string" method, effectively converting the visual number plate data into text format.

Deep Learning | Computer Vision

Cat and Dog Image Classification using CNN

  • Image Classification with CNNs: This project delves into the power of Convolutional Neural Networks (CNNs) for image classification tasks. CNNs utilize Conv2D layers with learnable filters to extract features like edges and textures from images, crucial for distinguishing cats from dogs.
  • Flattening and Architecture Design: Flattening layers transform the 2D feature maps into a 1D vector for feeding into fully-connected layers. The project explores designing a CNN architecture with appropriate stacking of these layers to achieve optimal classification performance.
  • Training Enhancements with Callbacks: Callbacks are introduced to monitor training progress and potentially adjust hyperparameters for better model performance.
  • Data Augmentation with ImageDataGenerator: This technique artificially expands the training dataset by applying random transformations (flips, rotations) to existing images, improving model generalizability and preventing overfitting.
  • Hands-on Learning: This project provides a practical approach to understanding CNNs. By implementing a CNN for cat and dog classification, you gain valuable experience with building and training image classification models.

NETFLIX MOVIE RECOMMENDATION SYSTEM

  • Developed a movie recommendation system for Netflix using various data preprocessing and machine learning techniques.
  • Merged multiple data files to create a comprehensive dataset, handled missing values (NaN), duplicates, and computed the distribution of ratings.
  • Calculated User-User Similarity matrix and Movie-Movie Similarity matrix using cosine similarity.
  • Engineered new features, such as similar user ratings for the same movie, similar movie ratings by the same user, average movie ratings, and average user ratings.
  • Modified the data to work seamlessly with Surprise-based algorithms from Surprise library.
  • Employed machine learning models, including XGBRegressor, Surprise BaselineModel, and Surprise KNNBaseline predictor, to find the best recommendation model.
  • Utilized Matrix Factorization-based algorithms, such as SVD and SVD++, for recommendation.
  • Implemented feature stacking by using the output of previous models as features for subsequent models, enhancing recommendation accuracy.
  • Considered RMSE (Root Mean Square Error) and MAPE (Mean Absolute Percentage Error) as evaluation metrics to assess the model's performance.
